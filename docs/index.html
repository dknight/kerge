About

kerge (light in Estonian🇪🇪) is a micro full-featured framework with a tiny footprint. kerge is based on design tokens. Combining just several tokens gives an almost limitless number of UI combinations.

Why?

I was really overwhelmed by massive-monstrous frameworks like Bootstap, MUI, Tailwind, etc. Usually I use only 5–10% of the features of these large CSS frameworks. I need a small framework that is easy to install and has only basic core features for UI prototyping. The solution is to use micro CSS frameworks; there are a huge number of options on the web like PureCSS, Lit, Chota, Skelet, etc. All of them are good, but... one of them is missing a feature, that another has, and vice versa. The authors of these frameworks usually do not maintain them. The last commit was about 3–5 years ago, and using outdated CSS techniques, bugs aren't fixed and aren't going to be fixed. This is why kerge was created. It has all the features I need, and I am going to maintain it until the end of my days.

Features

  • 🪶 Lightweight ~2kb (minified + gzipped)
  • 🚀 Easy to install and launch
  • 🚂 No dependencies
  • 🌐 Supports all modern browsers
  • 📐 Classic 12-columns grid
  • 🧱 Based on tokens
  • 🔩 Easy to customize with CSS custom properties
  • 🌙 Dark them included
  • 💬 Dialog and dropdown out-of-box
  • 🟩 Valid semantics
  • 👨‍👩‍👦‍👦 Accessible
  • 🖨️ Sustainable printing
